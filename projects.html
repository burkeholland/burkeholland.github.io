Beast Mode

Beast Mode is a custom chat mode for VS Code that turns your agent into a highly opinionated, agentic coding assistant. It builds on OpenAI's official 4.1 cookbook coding agent example and adds a workflow modeled after how a real developer works, with todo lists, deep planning, and recursive web research. Designed for GPT 4.1, but works well with Claude and other models.
