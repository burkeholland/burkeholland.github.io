Today I sat down to continue work on a Windows utlity app I've been building with Copilot to prove out some workflows that I think yield code that you can actually keep, isntead of building something 80% of the way and then just tossing all the code or forgetting about it. This is something that I do all the time so I know all too well that most code generated by AI today most likely never sees the light of day. As I watched the agent search files in my project to build up context as it was creating a plan document for a new feature, I had this familiar feeling. A still, small voice in my head that says "shouldn't you be doing something right now"? 

I spend a lot of my time these days watching agents work. I'm doing it right now, as I write this blog post. Just sitting here staring at the chat - read plan.md, searching for text, read PresetManager.cs. And I wonder - is this what its like to code now? I used to get lost in the sauce for hours on end writing code. My brain fully engaged and firing on all cylinders. If you're a dev, you know exactly what I'm talking about. "The Zone". It feels absolutely incredible to be in the zone. You could argue that being in the zone is the reason why we all do this job to begin with.
