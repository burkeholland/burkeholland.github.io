# Website Redesign Mockups

This directory contains 10 unique homepage design mockups for burkeholland.github.io.

## Viewing the Mocks

Open `index.html` in your browser to see an overview of all 10 designs with descriptions and links.

Each mock is a standalone HTML file that can be opened directly in any modern web browser.

## Design Philosophy

All mockups follow these principles:
- **Warm & Approachable**: No AI-looking designs or purple gradients
- **Dual Mode Support**: Each design adapts to light/dark mode via `prefers-color-scheme`
- **Unique Identity**: Every mock has its own typography, color palette, and personality
- **Content Focus**: Homepage layouts showcasing blog posts with mock data

## The 10 Designs

### 01. Warm Paper (Editorial)
Classic editorial layout inspired by print magazines. Cream and beige tones create a paper-like texture. Uses Crimson Pro serif for elegance.

### 02. Cozy Evening (Intimate)
Warm oranges and browns create an inviting, comfortable atmosphere. Card-based layout with soft shadows. Merriweather serif font.

### 03. Sunshine Minimal (Bright)
Clean and modern with yellow highlights for energy. High contrast design with Space Grotesk display font.

### 04. Forest Cabin (Natural)
Nature-inspired greens and wood tones for an organic feel. Bitter serif font with earthy palette.

### 05. Notebook (Brutalist)
Bold black and white with yellow highlights. Monospace typography (IBM Plex Mono) and high-contrast borders.

### 06. Coffee Shop (Comfortable)
Browns and tans creating a café atmosphere. Playfair Display serif with rounded corners and warm palette.

### 07. Twilight (Elegant)
Deep blues and indigos with subtle glows. Gradient backgrounds and Libre Baskerville serif.

### 08. Terracotta (Mediterranean)
Clay and earth tones with decorative lines. Cormorant serif font with warm terracotta colors.

### 09. Midnight Library (Dark Focus)
Dark mode-first design with deep navy and charcoal. Lora serif font with card elevations.

### 10. Campfire (Storytelling)
Orange and amber glow creating a warm storytelling atmosphere. Vollkorn serif with radial gradients.

## Technical Details

- **Framework**: Tailwind CSS via CDN
- **Fonts**: Google Fonts (unique per design)
- **Theming**: CSS custom properties for light/dark modes
- **Responsive**: Mobile-first layouts
- **Dependencies**: None (self-contained HTML files)
